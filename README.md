# Gator
This repository is for the [Blog Aggregator guided project](https://www.boot.dev/courses/build-blog-aggregator-golang) on [boot.dev](http://boot.dev). It is a CLI program built in Go.

# Requirements
In order to run Gator, you will need to install [Postgres](https://www.postgresql.org/) (v 15 or later) and [Go](https://go.dev/).

## - Installing Postgres
- macOS with [brew](https://brew.sh/)
    - `brew install postgresql@15`

- Linux WSL (Debian) [docs from Microsoft](https://learn.microsoft.com/en-us/windows/wsl/tutorials/wsl-database#install-postgresql)

    - `sudo apt update`
    - `sudo apt install postgresql postgresql-contrib`
